x2Polygons 0.0.5¶

This module calculates the distance between two matching polygons, such as building footprints. Package name is: x2polygons and can be downloaded from PyPi.

Download the package:

$ pip install x2polygons
Use cases:

This module could be used to compare reference building datasets with VGI datasets such as OSM.

Distance Functions:

  • Chamfer Distance

  • Hausdorff Distance

  • PoLis Distance [1].

  • Turn Function Distance [2].
